# 五线谱简谱学习小程序

这是一个基于微信小程序的五线谱简谱对照学习应用，帮助用户学习音乐理论知识。

## 功能特性

### 🎵 学习模式
- **练习模式**：自主练习，提升技能
- **PK模式**：双人对战，竞技比拼

### 🎹 核心功能
- 五线谱显示
- 简谱对照
- 钢琴键盘交互
- 实时反馈
- 学习统计

### 📊 统计功能
- 正确率统计
- 平均反应时间
- 答题进度跟踪
- PK模式对比

## 使用方法

### 开发环境设置
1. 下载并安装微信开发者工具
2. 在微信开发者工具中导入项目
3. 选择 `miniprogram` 目录作为项目根目录

### 运行项目
1. 在微信开发者工具中打开项目
2. 点击"编译"按钮
3. 在模拟器中预览效果

### 真机测试
1. 在微信开发者工具中点击"预览"
2. 使用微信扫描二维码
3. 在手机上体验小程序

## 主要组件说明

### Piano 组件
钢琴键盘组件，支持：
- 白键和黑键显示
- 键盘选择交互
- 正确答案高亮
- 触感反馈

### Staff 组件
五线谱显示组件，支持：
- 标准五线谱显示
- 音符位置定位
- 加线音符支持

### Jianpu 组件
简谱显示组件，支持：
- 简谱数字显示
- 显示/隐藏控制
- 动画效果

## 游戏逻辑

### 练习模式
1. 选择题目数量（15题/30题/无尽模式）
2. 系统随机生成音符
3. 用户在钢琴键盘上选择对应的键
4. 确认答案后显示简谱
5. 统计正确率和反应时间

### PK模式
1. 选择题目数量（15题/30题）
2. 玩家1先答题
3. 玩家1完成后切换到玩家2
4. 比较两位玩家的成绩
5. 显示获胜者

## 开发说明

本项目从 Vue.js 应用转换而来，保持了原有的功能特性，并针对微信小程序平台进行了优化。

### 主要转换内容
- Vue 组件 → 微信小程序自定义组件
- Vuex 状态管理 → 全局 globalData
- Vue Router → 小程序页面路由
- CSS → WXSS 样式

### 兼容性
- 支持微信小程序基础库 2.0+
- 兼容 iOS 和 Android 平台
- 适配不同屏幕尺寸
